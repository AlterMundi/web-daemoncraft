## 1. REBRANDING: HERMESCRAFT → DAEMONCRAFT

### Decisión de nombre
- Se descartó seguir con "HermesCraft" porque Hermes ya tiene asociaciones previas en el proyecto.
- Se eligió **DaemonCraft** por su triple resonancia:
  1. **Técnica:** En sistemas Unix, un daemon es un proceso persistente en segundo plano.
  2. **Mitología:** El daemon de Sócrates era un espíritu guía; Hermes conectaba mundos.
  3. **Gaming:** Resuena en Doom, Warhammer 40K, Diablo.

### Cambios realizados en la web
- Renombrado todo el proyecto a DaemonCraft
- Nueva paleta: Daemon Orange (#FF5722), Cyan Glow (#00E5FF), Void Black (#0B0C15)
- Tipografía: Fredoka (display), Nunito (body), Press Start 2P (UI gaming)
- Estética Minecraft pixelada con cielo animado, bloques, sol pixelado, nubes drift
- Landing page completa con:
  - Hero con daemon 3D (skinview3d)
  - 4 SOULs: Leñador, Arquitecta, Tutora, Guerrero
  - Timeline de onboarding (30 segundos)
  - Sección para padres (reportes Telegram)
  - Pricing: Free / Companion ($5) / Creator ($15) / School ($99)
  - Sección seed round / crowdfunding
- Componente SteveDaemon3D con skin-daemon.png

## 4. BOTS TELEGRAM

### Bot #1: Publicador Daemon
- Distribuye contenido a blog, redes sociales, newsletter
- Comandos: /post blog/x/ig/newsletter, /schedule, /metrics
- Stack: Python + python-telegram-bot + FastAPI

- Stack: Python + python-telegram-bot + LangChain + OpenAI

## 11. NOTAS Y OBSERVACIONES

- El rebrand de Hermes a Daemon fue consensuado por la fortaleza narrativa del término.
- La landing page usa Astro con React islands para el componente 3D.
- El sistema de memoria entre sesiones es el diferenciador clave frente a NPCs tradicionales.
- El ángulo educativo + supervisión parental es el nicho defensible contra competidores grandes.
- La "Primera Camada" es la narrativa de early adopters que construye comunidad desde el día 1.
- Los bots Telegram son infraestructura crítica tanto para operación como para comunidad.
- El dashboard es necesario para que el equipo (humano + IA) tenga visibilidad del estado del proyecto.
